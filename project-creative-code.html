Overview

A collection of generative art experiments built using the Canvas API and p5.js. Each piece explores a different concept — from particle systems and flow fields to recursive geometry and noise-driven motion.

The project started as a way to understand how mathematics and code intersect in visual art. Every sketch is written from scratch, treating the browser canvas as a blank sketchbook for computational expression.
