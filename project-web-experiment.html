Overview

A sandbox project for exploring unusual web interactions and CSS animation techniques. The focus was on pushing the limits of what's achievable with pure HTML and CSS — no heavy frameworks, just the raw primitives of the web platform.

Experiments include magnetic cursor effects, text scramble animations, smooth scroll-driven reveals, and layout transitions that challenge conventional grid thinking. Each one is a standalone page with a single focused idea.
