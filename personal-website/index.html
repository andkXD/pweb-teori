FAQ

How did you make the background effect?

The grain texture is made using p5.js — random noise points drawn each frame to simulate analog film grain.

What is this website for?

This is a personal portfolio website, created as a final project for a Web Programming course at university.

Can I use your code?

Feel free to learn from it, but please don't copy it as your own. Give credit if you're inspired by it.

What libraries do you use?

p5.js for the background animation. Google Fonts (Roboto Mono) for typography. Pure CSS for all layout and transitions.
