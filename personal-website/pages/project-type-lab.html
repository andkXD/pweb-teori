Overview

A laboratory for typographic experimentation — exploring variable fonts, kinetic type, and the expressive potential of letterforms in motion. Each piece treats type as both form and content simultaneously.

Experiments range from axis-manipulated variable font animations to generative type layouts driven by JavaScript. The goal is to push beyond type as mere text and into type as visual language.
