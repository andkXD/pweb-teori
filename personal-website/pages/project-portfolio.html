Overview

A personal portfolio website built from scratch as a final project for a Web Programming course. Inspired by the minimalist, typographic aesthetic of designers like Keita Yamada — where negative space and precision carry the design.

The background grain texture is generated in real-time using p5.js, layered with a vignette effect to create depth. Every layout decision was intentional — from the fixed corner positioning to the monospaced typeface that ties the whole thing together.

Role Design & Development
Stack HTML, CSS, JavaScript, p5.js
Duration 3 weeks
Status In progress
